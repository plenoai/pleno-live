## Even G2

The Even Hub plugin lives in [`apps/even-g2`](apps/even-g2). It uses the G2 microphone for live transcription and renders the result on the glasses without bundling a transcription API key.

**[Download Even G2 package](https://github.com/plenoai/pleno-live/releases/latest/download/pleno-live-g2.ehpk)**

```bash
pnpm even:g2:dev
pnpm even:g2:simulate
```
